# **App Name**: VizeTag Creator

## Core Features:

- Product Data Input Form: User interface for entering product details such as title, cash price, installment price, quantity, and dimensions (GEN, DER, YÜK).
- Real-time Label Preview: Displays a dynamic visual preview of the product label, instantly updating as form fields are changed.
- Dynamic Content Population: Automatically fills the preview label with the product title, cash and installment prices, and table details based on the form inputs.
- Fixed Branding Elements: Maintains the consistent display of the 'Vize Home' branded header and the payment method strip at the top and bottom of the label preview.
- Responsive Layout Management: Ensures an optimal two-panel layout (form on left, preview on right) on desktop, with responsive adjustments for other screen sizes.
- Text Adaptability: Intelligently handles varying lengths of product titles by automatically adjusting font size or breaking lines within the preview area to prevent overflow.
- Prominent Price Display: Ensures all price information is displayed in a large, clear, and highly readable format within dedicated price boxes and the product table.

## Style Guidelines:

- Primary color: A deep, professional burgundy-red (#72241D) for the brand's signature elements like headers and footers, conveying elegance and reliability.
- Background color: A subtle, warm off-white (#F4F1F0), nearly invisible, to provide a clean and professional canvas for the label content, visually akin to crisp paper.
- Accent color: A soft, muted rosy pink (#CC6691) for subtle highlights or secondary informational elements, offering a harmonious contrast to the primary brand color.
- Headline and price font: 'Poppins' (geometric sans-serif) for its contemporary precision and impact, ensuring titles and prices are clear and commanding.
- Body and form input font: 'Inter' (grotesque sans-serif) for exceptional legibility and a neutral, modern aesthetic across smaller text, form labels, and table content.
- Clean, modern line icons for product categories (e.g., simple sofa icon placeholder) and recognizable payment provider logos to maintain a professional, minimalist look.
- A responsive two-panel desktop layout with the form on the left and the A4 portrait-ratio label preview on the right. Preview elements like the 'Vize Home' top brand area and bottom payment strip are fixed, maintaining consistent visual identity.
- Subtle, fluid transitions for text updates and dynamic content changes within the preview, ensuring a smooth and refined user experience without distracting movements.
